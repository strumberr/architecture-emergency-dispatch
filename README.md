## Summary
A system where citizens can report emergencies, dispatchers can assign responders, and responders receive alerts and update their status.

## User Roles

- **Citizen:** reports issues (accident, fire, medical, etc.)
- **Dispatcher:** monitors reports and assigns responders
- **Responder:** receives tasks, updates progress
